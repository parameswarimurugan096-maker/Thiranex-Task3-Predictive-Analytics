# Predictive Analytics Using Historical Data

## Objective
Build a predictive model using historical sales data to forecast future sales trends.

## Tools Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Jupyter Notebook

## Dataset
E-commerce Sales Dataset

## Process
1. Loaded and cleaned historical sales data
2. Converted order dates into numerical values
3. Applied Linear Regression model
4. Predicted future sales trends
5. Visualized predictions using charts

## Output Files
- sales_prediction.png
- sales_predictions.csv

## Outcome
Successfully developed a predictive analytics model to forecast future sales based on historical data.
